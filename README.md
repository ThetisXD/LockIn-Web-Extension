<h1 align="left">LockIn Web Extension</h1>

###
<h2 align-"left">✨ What is it?</h2>
<p align="left">This web extension is a productivity time management tool with tab restrictions, designed to enhance user productivity and reduce distractions.</p>

###

<h2 align-"left">✨ What does it do?</h2>
<p align="left">The LockIn Web Extension allows users to customize how they set up time, with three modes: standard countdown timer, pomodoro, and standard clock. In addition, LockIn has tab blacklists features that allow users to blacklist and prevent users from accessing certain blacklist sites.</p>

###
<h4 align="left">⌛ 1. Countdown</h4>
<p align="left">the user set up the time, and the timer will count down until it hit 0.</p>

###

<h4 align="left">↔️ 2. Pomodoro</h4>
<p align="left">The user set up the duration for resting state, alongsides user producitivty session. The timer will alternates between resting and productivity for the user</p>

###

<h4 align="left">⏰ 3. Standard Clock</h4>
<p align="left">In this mode, the timer display user current military time by default or PM/AM format if enabled.</p>

###

<h2 align-"left">✨ Tab Blacklist feature</h2>
<p align="left">The web extension scans the user current browser and display all current avaliable tab. After, the user can select which tab they do not want to be blacklisted, any blacklisted tabs will automatically be restricted, prevent the user from accessing certain site. </p>





<h2 align-"left">✨privacy and permissions</h2>


<h4 align="left">Purpose</h4>
<ul align="left">
  <li>LockIn only records the tab URL string, and the tab corresponding favicon if avaliable.</li>
  <li>The tab permission is used to store the tab URL string and the favicon in the browser's local storage for processing.</li>
  <li>The tab blacklist is an optional feature that users can enable. When active, the web browser periodically scans the browser and matches the web browser URL to it, blocking it from the browser.</li>
</ul>

<h4 align="left">privacy policy</h4>
<ul align="left">
  <li>LockIn web extension do not records user sensitive content, form data and browsing history</li>
  <li>The tab permission is strictly require for blacklist tabs to be operational</li>
  <li>The storage permission is only used to stored system essential functions and blacklists array.</li>
</ul>
