<h1 align="left">LockIn Web Extension</h1>

###
<h2 align-"left">⭐ What is it?</h2>
<p align="left">This web extension is a productivity time management tool with tab restrictions, designed to enhance user productivity and reduce distractions.</p>

###

<h2 align-"left">✨ What does it do?</h2>
<p align="left">The LockIn Web Extension allows users to customize how they set up time, with three modes: standard countdown timer, pomodoro, and standard clock. In addition, LockIn has tab blacklists features that allow users to blacklist and prevent users from accessing certain blacklist sites. The timer and tab blacklist will continue to runs even if the sidepanel is close.</p>


###
<h4 align="left">⌛ 1. Countdown</h4>
<p align="left">the user set up the time, and the timer will count down until it hit 0.</p>

###

<h4 align="left">↔️ 2. Pomodoro</h4>
<p align="left">The user set up the duration for resting state, alongsides user producitivty session. The timer will alternates between resting and productivity for the user</p>

###

<h4 align="left">⏰ 3. Standard Clock</h4>
<p align="left">In this mode, the timer display user current military time by default or PM/AM format if enabled.</p>

###

<h2 align-"left">🚫 Tab Blacklist feature</h2>
<p align="left">The web extension scans the user's current browser and displays any blacklisted tabs the user has last saved. If not available, the system displays all current available tabs. After that, the user can select which tab they do not want blacklisted. Any blacklisted tabs will automatically be restricted, preventing the user from accessing certain sites, and the user can reset the blacklist and start over </p>



<h2 align="left">✨ How to install?</h2>
<ul align="left">
  <li>1. Open the web extension on either Google chrome, Microsoft Edge, or Opera. </li>
  <li>2. On the search bar, depends on your browser, type either one of the following.</li>
  <ul align="left">
      <li>Chrome: chrome://extensions</li>
      <li>Edge: edge://extensions</li>
      <li>Opera: opera://extensions</li>
  </ul>
  <li>3. Turn on the developer mode and import unloaded package</li>
  <li>4. Click the web extension icon and click the side panel to turn on LockIn Web Extension</li>
</ul>

<h2 align="left">✨privacy and permissions disclaimer</h2>


<h4 align="left">Purpose</h4>
<ul align="left">
  <li>LockIn only records the tab URL string, and the tab corresponding favicon if avaliable.</li>
  <li>The tab permission is used to store the tab URL string and the favicon in the browser's local storage for processing.</li>
  <li>The tab blacklist is an optional feature that users can enable. When active, the web browser periodically scans the browser and matches the browser URL to it, and closing the tab effectively prevents the user from accessing the blacklist tabs.</li>
</ul>

<h4 align="left">privacy policy</h4>
<ul align="left">
  <li>LockIn web extension do not records user sensitive content, form data and browsing history</li>
  <li>The tab permission is strictly require for blacklist tabs to be operational</li>
  <li>The storage permission is used to stored system essential functions and blacklists array.</li>
  <li>The storage permission is used to continuously track the timer and tab blacklist monitor, allowing the user to close and reopen the side panel without resetting progress.</li>
</ul>
